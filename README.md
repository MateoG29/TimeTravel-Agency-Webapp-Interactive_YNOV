## Présentation du Projet

Réalisation d'une plateforme moderne pour l'agence **TimeTravel Agency**, permettant de découvrir trois destinations historiques majeures, d'interagir avec un conseiller IA et de simuler une réservation sécurisée malgré les aléas de l'histoire.

---

## Phase 1 : Architecture & Planning

### Features Implémentées :

* **Landing Page :** Interface en "Dark Mode" premium avec accents dorés.
* **Hero Section :** Fond animé évoquant le flux temporel avec bouton CTA "Explorer les Époques".
* **Galerie interactive :** 3 époques (Paris 1889, Crétacé, Florence 1504).
* **Agent Conversationnel :** Chatbot intégré pour le conseil client.
* **Formulaire de Réservation :** Système de planification avec validation.

---

## Phase 2 : Génération de Code & Assets

### Stack Technique :

* **Framework :** React + Tailwind CSS
* **Outil de Vibe Coding :** Bolt.new (moteur Claude 3.5 Sonnet)
* **Animations :** Framer Motion (Transitions fluides et hover effects sur les cards)
* **Hébergement des Assets :** https://bolt.new/

### Intégration Visuelle :

* Cartes interactives avec zoom au survol et glassmorphism.
* Intégration des visuels générés lors de la Session 1 (Paris, Crétacé, Florence).

---

## Phase 3 : Intelligence Artificielle & Agents

### Agent Conversationnel :

* **Identité :** Assistant expert de l'agence TimeTravel.
* **Capacités :** Répond sur les prix, les risques (microbes au Crétacé) et l'histoire (Expo Universelle).

### Optionnel - Personnalisation :

* **Quiz Interactif :** Système de recommandation en 3 questions pour définir la destination idéale du client (Nature vs Culture).
